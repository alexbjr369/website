## Lerna

- **[`lerna`](https://www.npmjs.com/package/lerna)**

```bash
yarn add -WD lerna
```

##### Configuration

```json {1-21} title="lerna.json"
{
  "npmClient": "yarn",
  "useWorkspaces": true,
  "version": "independent",
  "packages": [
    "packages/*"
  ],
  "command": {
    "version": {
      "message": "chore(release): publish"
    },
    "publish": {
      "registry": "http://localhost:4873/",
      "ignoreChanges": [
        "**/*.md"
      ]
    }
  }
}
```

## Commitlint/Lerna Integration

- **[`@commitlint/config-lerna-scopes`](https://www.npmjs.com/package/@commitlint/config-lerna-scopes)**

```bash
yarn add -WD @commitlint/config-lerna-scopes
```

##### Configuration

```javascript {1-21} title="commitlint.config.js"
const {
  utils: { getPackages },
} = require('@commitlint/config-lerna-scopes');

module.exports = {
  extends: [
    '@commitlint/config-conventional',
    '@commitlint/config-lerna-scopes',
  ],
  rules: {
    'scope-enum': async (ctx) => [
      2,
      'always',
      [
        ...(await getPackages(ctx)),
        // insert custom scopes below:
        'release',
      ],
    ],
  },
};
```
