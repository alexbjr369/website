## ESLint

- **[`eslint`](https://www.npmjs.com/package/eslint)**

```bash
# npm
npm i -D eslint

# yarn
yarn add -D eslint
```

##### Plugins

- **[`eslint-config-airbnb-base`](https://www.npmjs.com/package/eslint-config-airbnb-base)**
- **[`eslint-config-prettier`](https://www.npmjs.com/package/eslint-config-prettier)**
- **[`eslint-plugin-import`](https://www.npmjs.com/package/eslint-plugin-import)**
- **[`eslint-plugin-prettier`](https://www.npmjs.com/package/eslint-plugin-prettier)**

```bash
# npm
npm i -D eslint-config-airbnb-base eslint-config-prettier eslint-plugin-import eslint-plugin-prettier

# yarn
yarn add -D eslint-config-airbnb-base eslint-config-prettier eslint-plugin-import eslint-plugin-prettier
```

##### Configuration

```json title=".eslintrc.json"
{
  "env": {
    "es2022": true,
    "browser": true
  },
  "extends": ["airbnb-base", "prettier"],
  "plugins": ["prettier"],
  "rules": {
    "prettier/prettier": "error"
  }
}
```

```json {10} title="package.json"
{
  "name": "project",
  "version": "1.0.0",
  "main": "index.js",
  "license": "MIT",
  "scripts": {
    "prepare": "npx husky install",
    "prettier": "prettier . --write --ignore-unknown",
    "commit": "git-cz",
    "eslint": "eslint **/*.js --fix"
  },
  "lint-staged": {
    "**/*": "npm run prettier",
  },
  "config": {
    "commitizen": {
      "path": "./node_modules/cz-conventional-changelog"
    }
  }
}
```

## lint-staged

##### Configuration

```json {14} title="package.json"
{
  "name": "eslint-javascript",
  "version": "1.0.0",
  "description": "",
  "main": "index.js",
  "scripts": {
    "prepare": "npx husky install",
    "prettier": "prettier . --write --ignore-unknown",
    "commit": "git-cz",
    "eslint": "eslint **/*.js --fix"
  },
  "lint-staged": {
    "**/*": "npm run prettier",
    "**/*.js": "npm run eslint"
  },
  "config": {
    "commitizen": {
      "path": "./node_modules/cz-conventional-changelog"
    }
  }
}
```
