## Stylelint

- **[`stylelint`](https://www.npmjs.com/package/stylelint)**

```bash
# npm
npm i stylelint -D

# yarn
yarn add stylelint -D
```

##### Plugins

- **[`stylelint-config-prettier`](https://www.npmjs.com/package/stylelint-config-prettier)**
- **[`stylelint-config-rational-order`](https://www.npmjs.com/package/stylelint-config-rational-order)**
- **[`stylelint-order`](https://www.npmjs.com/package/stylelint-order)**
- **[`postcss-scss`](https://www.npmjs.com/package/postcss-scss)**

```bash
# npm
npm i stylelint-config-prettier stylelint-config-rational-order stylelint-order postcss-scss -D

# yarn
yarn add stylelint-config-prettier stylelint-config-rational-order stylelint-order postcss-scss -D
```

##### Configuration

```json title=".stylelintrc.json"
{
  "extends": ["stylelint-config-rational-order", "stylelint-config-prettier"],
  "customSyntax": "postcss-scss"
}
```

```json {10} title="package.json"
{
  "name": "project",
  "version": "1.0.0",
  "main": "index.js",
  "license": "MIT",
  "scripts": {
    "prepare": "npx husky install",
    "prettier": "prettier . --write --ignore-unknown",
    "commit": "git-cz",
    "stylelint": "stylelint **/*.scss --fix"
  },
  "lint-staged": {
    "**/*": "npm run prettier"
  },
  "config": {
    "commitizen": {
      "path": "./node_modules/cz-conventional-changelog"
    }
  }
}
```

## Format on Save

```json {7} title=".vscode/settings.json"
{
  "editor.codeActionsOnSave": {
    "source.fixAll": true
  },
  "editor.defaultFormatter": "esbenp.prettier-vscode",
  "editor.formatOnSave": true,
  "stylelint.validate": ["scss"]
}
```

## lint-staged

##### Configuration

```json {14} title="package.json"
{
  "name": "project",
  "version": "1.0.0",
  "main": "index.js",
  "license": "MIT",
  "scripts": {
    "prepare": "npx husky install",
    "prettier": "prettier . --write --ignore-unknown",
    "commit": "git-cz",
    "stylelint": "stylelint **/*.scss --fix"
  },
  "lint-staged": {
    "**/*": "npm run prettier",
    "**/*.scss": "npm run stylelint"
  },
  "config": {
    "commitizen": {
      "path": "./node_modules/cz-conventional-changelog"
    }
  }
}
```
