## [Short Circuiting (&& and ||)](https://www.udemy.com/course/the-complete-javascript-course/learn/lecture/22648547#overview)

Logical Operators:
- Can use any data type
- Can return any data type
- short-circuiting (short-cut evaluation)

```js title="script.js"
'use strict';

// if first element is truthy, the second element will not be evaluated
console.log(3 || 'John'); // result: 3
console.log('' || 'John'); // result: 'John'
console.log(true || 0); // result: true
console.log(undefined || null); // result: null

console.log(undefined || 0 || '' || 'John' || 5); // result: 'John'

const guests2 = restaurant.numGuests || 10; // result: 10
console.log(guests); // result: 10
```

```js title="script.js"
'use strict';

// if first element is falsy, the second element will not be evaluated
console.log(0 && 'John'); // result: 0
console.log(5 && 'John'); // result: 'John'

console.log('John' && 5 && undefined && 0 && ''); // result: undefined
```

'use strict';

// falsy or truth

const restaurant = {
