## [Configurando Babel](https://app.rocketseat.com.br/node/chapter-i-1/group/configurando-ambiente/lesson/configurando-babel-2)

**Installing Babel/React.js Ecosystem**

- **[`@babel/core`](https://www.npmjs.com/package/@babel/core)**
- **[`@babel/cli`](https://www.npmjs.com/package/@babel/cli)**
- **[`@babel/preset-env`](https://www.npmjs.com/package/@babel/preset-env)**
- **[`@babel/preset-react`](https://www.npmjs.com/package/@babel/preset-react)**

```bash
# npm
npm i -D @babel/core @babel/cli @babel/preset-env @babel/preset-react

# yarn
yarn add -D @babel/core @babel/cli @babel/preset-env @babel/preset-react
```

**Configuring Babel/React.js**

```txt {2} title=".gitignore"
node_modules
dist
```

```javascript {1-3} title="babel.config.js"
module.exports = {
  presets: ['@babel/preset-env', '@babel/preset-react'],
};
```

```javascript {1-5} title="src/index.jsx"
import React from 'react';

function App() {
  return <h1>Hello World</h1>;
}
```

**Running Babel**

```bash
# npm
npx babel src/index.jsx --out-file dist/bundle.js

# yarn
yarn babel src/index.jsx --out-file dist/bundle.js
```
