## [Estado do componente](https://app.rocketseat.com.br/node/chapter-i-1/group/conceitos-importantes/lesson/estado-do-componente)

```javascript {1-18} title="src/components/Counter.jsx"
import { useState } from 'react';

export function Counter() {
  const [counter, setCounter] = useState(0);

  function increment() {
    setCounter(counter + 1);
  }

  return (
    <div>
      <h2>{counter}</h2>
      <button type="button" onClick={increment}>
        Increment
      </button>
    </div>
  );
}
```

```javascript {1,7-10} title="src/App.jsx"

    </>

}
```

## [A imutabilidade no React](https://app.rocketseat.com.br/node/chapter-i-1/group/conceitos-importantes/lesson/a-imutabilidade-no-react)

**Mutabilidade**

```javascript
const usuarios = ['a', 'b', 'c'];
usuarios.push('d'); // altera o valor da variavel 'usuarios' na memória
```
**Imutabilidade**

```javascript
const usuarios = ['a', 'b', 'c'];
const novoUsuarios = [...usuarios, 'd']; // cria um novo espaço na memória contendo a nova informação
```

## [Fast Refresh no Webpack](https://app.rocketseat.com.br/node/chapter-i-1/group/conceitos-importantes/lesson/fast-refresh-no-webpack)

Mantém o estado dos componentes no hot reload do webpack.

- **[`@pmmmwh/react-refresh-webpack-plugin`](https://www.npmjs.com/package/@pmmmwh/react-refresh-webpack-plugin)**
- **[`react-refresh`](https://www.npmjs.com/package/react-refresh)**

```bash
# npm
npm i @pmmmwh/react-refresh-webpack-plugin react-refresh -D

# yarn
yarn add @pmmmwh/react-refresh-webpack-plugin react-refresh -D
```

```javascript {3,20,23,27,33-40} title="webpack.config.js"
const path = require('path');
const HtmlWebpackPlugin = require('html-webpack-plugin');
const ReactRefreshWebpackPlugin = require('@pmmmwh/react-refresh-webpack-plugin');

const isDevelopment = process.env.NODE_ENV !== 'production';

module.exports = {
  mode: isDevelopment ? 'development' : 'production',
  devtool: isDevelopment ? 'eval-source-map' : 'source-map',
  entry: path.resolve(__dirname, 'src', 'index.jsx'),
  output: {
    path: path.resolve(__dirname, 'dist'),
    filename: 'bundle.js',
  },
  resolve: {
    extensions: ['.js', '.jsx'],
  },
  devServer: {
    static: path.resolve(__dirname, 'public'),
    hot: true,
  },
  plugins: [
    isDevelopment && new ReactRefreshWebpackPlugin(),
    new HtmlWebpackPlugin({
      template: path.resolve(__dirname, 'public', 'index.html'),
    }),
  ].filter(Boolean),
  module: {
    rules: [
      {
        test: /\.jsx$/,
        exclude: /node_modules/,
        use: {
          loader: 'babel-loader',
          options: {
            plugins: [
              isDevelopment && require.resolve('react-refresh/babel'),
            ].filter(Boolean),
          },
        },
      },
      {
        test: /\.scss$/,
        exclude: /node_modules/,
        use: ['style-loader', 'css-loader', 'sass-loader'],
      },
    ],
  },
};
```
