## [Configurando fonte externa](https://app.rocketseat.com.br/node/fundamentos-do-next-js/group/estrutura-da-aplicacao-2/lesson/configurando-fonte-externa)

- **_app is loaded everytime a page is changed**.
- **_document is loaded only once (similar to public/index.html)**

```typescript title="src/pages/_document.tsx"
import Document, { Html, Head, Main, NextScript } from 'next/document';

export default class MyDocument extends Document {
  render() {
    return (
      <Html>
        <Head>
          <link rel="preconnect" href="https://fonts.googleapis.com" />
          <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
          <link
            href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;700;900&display=swap"
            rel="stylesheet"
          />
          <title>ig.news</title>
        </Head>
        <body>
          <Main />
          <NextScript />
        </body>
      </Html>
    );
  }
}
```
