## [Assignment 1: Practicing Components](https://www.udemy.com/course/the-complete-guide-to-angular-2/learn/practice/266#overview)

**1. Create two new Components (manually or with CLI): WarningAlert and SuccessAlert:**

```bash
# npm
npx ng g c warning-alert

# yarn
yarn ng g c warning-alert
```

```bash
# npm
npx ng g c success-alert

# yarn
yarn ng g c success-alert
```

**2. Output them beneath each other in the AppComponent:**

```html {1-8} title="src/app/app.component.html"
<div class="container">
  <div class="row">
    <div class="col-xs-12">
      <app-warning-alert></app-warning-alert>
      <app-success-alert></app-success-alert>
    </div>
  </div>
</div>
```

**3. Output a warning or success message in the Components:**

```html {1-8} title="src/app/app.component.html"
<div class="container">
  <div class="row">
    <div class="col-xs-12">
      <app-warning-alert></app-warning-alert>
      <app-success-alert></app-success-alert>
    </div>
  </div>
</div>
```

```html {1} title="src/app/success-alert.component.html"
<p>This is a warning, you are in danger!</p>
```

```html {1} title="src/app/warning-alert.component.html"
<p>You are so successful!</p>
```

**4. Style the Components appropriately (maybe some red/green text?):**

```scss {1-5} title="src/app/success-alert.component.scss"
p {
  padding: 20px;
  background-color: palegreen;
  border: 1px solid green;
}
```

```scss {1-5} title="src/app/warning-alert.component.scss"

  padding: 20px;
  background-color: mistyrose;
  border: 1px solid red;
}
```
